The DOM is known as the Document object model and allows the accessing of the documents like HTML. It is a programming interface that allows us to create, change, or remove elements from the document. We can also add events to these elements to make our page more dynamic.

There are 5 ways to select DOM elements.
  1. getElementsByTagName()
  2. getElementsByClassName()
  3. getElementById()
  4. querySelector()
  5. querySelectorAll()

The Event bubbling in JavaScript is a type of event propagation. The event triggers the innermost target element and consecutively triggers the parent element of the target element in the same hierarchy until it triggers the outermost element.

A callback function in javascript is a function that is passed as an argument in another function. Which is then called inside the parent function to complete a routine or an action.Callback functions are needed because many JavaScript actions are asynchronous.
